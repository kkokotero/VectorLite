# Code of Conduct

## Our commitment

We want VectorLite to be a welcoming project for everyone.

We expect contributors, users, and maintainers to treat each other with respect.

## Expected behavior

- Be constructive.
- Be kind and direct.
- Focus on the code, not the person.
- Give feedback that helps others improve.
- Respect different levels of experience.

## Unacceptable behavior

- Harassment
- Discrimination
- Personal attacks
- Trolling
- Intimidation
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject contributions that violate this code of conduct.

If you need to report a problem, open a private issue or contact the project maintainers through the repository’s communication channels.

## Scope

This code of conduct applies to all project spaces, including:

- issues
- pull requests
- discussions
- commit messages
- documentation
